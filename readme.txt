== Description ==

A clean, modern WordPress block theme built for GatherPress. Features event-first homepage layouts, venue pages with maps, and a refined design system using Inter font and an indigo accent palette.

This theme is designed for any community using GatherPress to manage events.

== Features ==

* Full Site Editing (FSE) block theme
* Event-focused homepage with upcoming and past event grids
* Single event and venue templates with sidebar details
* Custom comment form with character counting
* Sticky header with smooth transitions
* Event search functionality
* GatherPress admin integration for theme settings

== Installation ==

1. Install and activate the **GatherPress** plugin
2. Upload this theme into /wp-content/themes/gatherpress-org-theme/
3. Activate the theme

== Frequently Asked Questions ==

= Does this theme work without GatherPress? =
The theme will activate, but event-specific features require the GatherPress plugin.
