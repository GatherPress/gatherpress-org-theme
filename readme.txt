== Description ==

This theme extends the GatherPress theme with custom features such as:
- Signup with Email OTP verification
- Custom login form
- Custom comment form with validation
- Releases custom post type
- Custom taxonomy for releases
- Additional block styles and enqueue improvements

This theme is designed for development, contribution, and testing.
"GatherPress" is open source software. The following people have contributed to this plugin. The names link
to WordPress profiles.
== Features ==

* Parent theme: GatherPress
* Child theme integrations
* Custom Signup Form (OTP)
* Custom Login Form
* Custom Comment Form
* Additional block styles
* Releases CPT
* Release Category taxonomy
* Swiper slider support

== Installation ==

1. Install the parent theme **GatherPress**
2. Upload this theme into /wp-content/themes/gatherpress-child/
3. Activate the theme
4. Add shortcodes in pages for custom forms if needed

== Frequently Asked Questions ==

= Does this theme work independently? =
No, you must install the GatherPress parent theme.

= Are shortcodes supported? =
Yes: `[custom-login-form]`, `[custom-signup-form]`, `[custom_comment_form]`.
